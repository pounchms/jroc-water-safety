How the risk score is calculated

A rule-based score, 1–10: a base level from the current river height, plus bonus points for specific danger patterns. Nothing here is a black box — every point is traceable to a rule below.

1 — Base score, from gage height at Richmond

Under 4 ft1Low
4–8 ft3Moderate
8–14 ft5High
14 ft+6Flood

Base + bonuses, capped at 10 → the risk index shown above

These breakpoints are defensible starting estimates, not yet validated against real incident data (Richmond Fire's incident records aren't available to this project yet — see the footer). Treat the score as a rule-based estimate, not an official forecast.

If you or someone else is in danger on the river right now, call 911. This page is an informational tool, not an emergency alert system, and is not monitored in real time by any agency.

Data sources: USGS gauge 02037500 (Richmond) and 02026000 (Bent Creek), plus NWS weather observations. Refreshed daily via automated pipeline.

The risk score is a rule-based estimate built by a student team, not an official flood or safety forecast. For the National Weather Service's official river forecast, see NWS AHPS. For general river safety guidance, see Friends of James River Park.
